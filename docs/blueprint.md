# **App Name**: UPSA Marketplace

## Core Features:

- User Authentication: login and signup pages, which uses fire base for auth, and creates user's unique profile
- Landing Page: Landing page with links to login and signup pages.
- Browse Products: Display all uploaded goods and services with sort options.
- Shopping Cart: Add items to cart while browsing.
- User Profiles: Auto set up profiles when loged in, with user-specific info stored using Firebase.
- Product Upload and Payment: Enable users to upload and sell products with payment via Paystack.
- Collect Upload Fee: Collect an 8 GHC fee for each upload.
- AI Product Boost: AI tool analyzes listings and user activity to determine if and when to boost visibility. Boost products in 'buy.html' for a set period upon 2 GHC extra payment.
- Sales and Purchase Tracking: Track sales and purchases from both 'sell.html' and 'buy.html'.
- Payment Handling: payments for sales product upload to be handled by paystack(providing a placeholer for api keys in the code)

## Style Guidelines:

- Primary color: Deep sky blue (#3299bb) - conveys trust, security, and reliability, which is crucial for a marketplace.
- Background color: Light blue (#E0F7FA) - a soft, desaturated tone derived from the primary, providing a clean and calming backdrop.
- Accent color: Coral (#FF7F50) - to draw attention to important elements and calls to action
- Body and headline font: 'PT Sans', a modern, legible, humanist sans-serif that works well for both.
- Use simple, clear icons for navigation and product categories.
- Design a responsive layout optimized for mobile devices, following a card-based design for product listings.
- Incorporate subtle transitions and animations to enhance user experience without being intrusive.
